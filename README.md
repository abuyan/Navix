# Nivix - 灵犀

AI 驱动的智能书签，懂收藏，更懂整理。

## 项目简介
Nivix 是一个现代化的导航站和书签管理工具，旨在利用 AI 技术帮助用户自动整理、分类和丰富收藏的内容。

## 功能特性
- **AI 智能整理**：自动提取网站标题、描述和图标。
- **多维度视图**：支持不同的导航版块和分类。
- **现代化 UI**：拥有毛玻璃效果、深色模式和流畅的交互动画。
- **隐私优先**：数据本地优先，安全可控。
