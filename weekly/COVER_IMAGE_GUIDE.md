# 周刊封面图使用说明

## 如何添加封面图

在 Markdown 文件的 frontmatter 中添加 `cover` 字段:

```markdown
---
title: "2026年第1周周刊"
date: "2026-01-06"
category: "2026年存档"
tags: ["AI", "技术", "工具"]
excerpt: "新年第一周,分享一些有趣的AI工具和技术趋势"
cover: "/images/weekly/2026-01-cover.jpg"
---
```

## 封面图规格建议

- **尺寸**: 建议 800x450 像素 (16:9 比例)
- **格式**: JPG, PNG, WebP
- **位置**: 放在 `public/images/weekly/` 目录下
- **文件名**: 使用有意义的命名,如 `2026-01-cover.jpg`

## 示例

### 有封面图的文章
文章列表中会在右侧显示封面图缩略图

### 没有封面图的文章
不会显示任何占位符,保持简洁布局

## 注意事项

1. 封面图路径必须以 `/` 开头(相对于 public 目录)
2. 确保图片文件存在,否则会显示破损图片
3. 建议使用压缩后的图片以提高加载速度
